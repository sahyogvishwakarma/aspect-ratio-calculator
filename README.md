# Aspect Ratio Calculator

A lightweight online tool to calculate and resize image dimensions while preserving exact aspect ratio.

## Features

- Upload image to auto-detect dimensions
- Resize with max side constraint
- Resize with total pixel constraint
- Dark / Light mode
- Fully client-side (no backend)
- Works on GitHub Pages
