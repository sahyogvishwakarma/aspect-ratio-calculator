How to Use

  • Reference Image: Upload an image to auto-fill the width and height fields.
  • Width & Height: You can manually enter width and height if you don't want to use an image.
  • Max Pixels Per Side: Optional. Enter the maximum number of pixels allowed for the width or height. The tool will adjust the other dimension to maintain the aspect ratio.
  • Max Total Pixels: Optional. Enter the maximum total number of pixels (width × height). The tool will scale the image to fit within this limit, maintaining the aspect ratio.
  • Live Calculation: The tool updates automatically as you type or upload an image—no need to click a button.
  • Output: The calculated width, height, total pixels, and aspect ratio are displayed below the inputs.
  • Constraints: If any constraints are applied, they will be shown in the “Constrained by” badge below the output.
  • Notes: The tool preserves the aspect ratio exactly. It will also allow upscaling if your maximum pixel constraints allow it.
